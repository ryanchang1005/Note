好看log
git log --all --decorate --oneline --graph

顯示差異
git diff <filename>

勿加到暫存區(add), 取消
git reset <filename>

強制還原上一個版本(本次修改(unstashed)將會回溯)(危)
git reset --hard

設定縮寫
git config --global alias.lg "log --all --decorate --oneline --graph"
git config --global alias.cm commit
git config --global alias.st status
git config --global alias.br branch
git config --global alias.ch checkout

Git設定檔
~/.gitconfig

clone時把submodule也抓下來
git clone --recursive xxx.git

事後才知道有submodule, 事後抓
git submodule init
git submodule update --recursive

顯示目前remote的url
git remote -v

改remote url
git remote set-url origin https://USERNAME@gitlab.com/OOXX/OOXX/OOXX.git

新增branch並checkout過去
git checkout -b new-branch

刪除branch
git branch -d new-branch
