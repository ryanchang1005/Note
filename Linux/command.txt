顯示容量
du -sh ~/

硬碟使用率
df -h

自訂命令別名
vi ~/.bashrc
ex:
    alias dc=docker-compose
    alias d=docker
    dexec(){
        docker exec -it "$1" /bin/bash
    }

找出佔用 8000 port的process
lsof -i:8000

清除pid為12345的process
kill 12345

匹配文字
grep 關鍵字 檔案
ls | grep .txt  // 搭配pipe找出檔案名稱中含有".txt"的檔案
grep --color=always -n 123 *.txt  // 顏色標示
grep "^a" data.txt  // a開頭
grep "b$" data.txt  // b結尾

替代文字
sed -i 's/old_text/new_text/g' file.txt
